CodeKata > Hangman

The game is initialised with a word that is hidden from the player.

Guesses are made for letters that the word may contain.

If the guessed letter is in the word, all instances of it are revealed.

If the guessed letter is not in the word, a penalty is added.

If the maximum number of penalties is reached, the game is lost.

If all letters in the word are revealed, the game is won.

Implement a hangman game.
