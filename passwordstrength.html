CodeKata > PasswordStrength

Determine whether password string is valid based on rules.

The following rules can be defined:

  • Length greater than given number of characters.
  • Contains at least a given number of uppercase letters.
  • Contains at least a given number of lowercase letters.
  • Contains at least a given number of digits.
  • Contains at least a given number of certain special characters e.g. !, $, %, ^, &, *, or #.

A given number of these rules must be met for a password to be valid.

One or more of the rules can also be mandatory and must be met regardless.
