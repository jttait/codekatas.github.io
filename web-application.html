Create publicly accessible API to convert between Celsius, Kelvin, and Fahrenheit temperatures.

C = (5 / 9) * (F - 32)

K = C + 273.15

Request example: HTTP POST { "value": 100, "scale": "celsius" }

Response example: HTTP 200 { "celsius": 100, "kelvin": 373.15, "fahrenheit": 212 }
