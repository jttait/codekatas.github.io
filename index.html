CodeKata

  • PokerHands: Score poker hands.
  • FizzBuzz: First 100 FizzBuzz numbers.
  • TenPinBowling: Score ten pin bowling.
  • ShoppingBasket: Price a shopping basket.
  • IntegerToWords: Convert integer to words.
  • MnistClassification: Classify the MNIST images into digits.
  • WordWrap: Wrap text to a maximum line length.
  • HelloWorld: Print "hello, world."
  • BinarySearch: Implement binary search.
  • Quicksort: Implement quicksort.
  • PasswordStrength: Check if a password meets the criteria.
  • MarsRover: Execute list of commands to move rover.
