CodeKata > StringCalculator

The input is an equation expressed as a string e.g. "2+4*3".

The possible operators, in order of precedence, are:

  • Parenthesis ()
  • Exponents ^
  • Division /
  • Multiplication *
  • Addition +
  • Subtraction -

Determine the result of the calculation e.g. 24 for the example input above.
